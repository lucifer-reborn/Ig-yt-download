# LUCIFER Downloader

Terminal-based YouTube & Instagram downloader.

## Features
- YouTube
  - Audio (MP3)
  - Video (MP4)
- Instagram
  - Audio (MP3)
  - Reel (MP4)
  - Post images (JPG)

## Folder Structure
BY-LUCIFER/
├── YOUTUBE/
│   ├── AUDIO/
│   └── VIDEO/
└── INSTAGRAM/
    ├── AUDIO/
    ├── VIDEO/
    └── POST/

## Requirements
- Python 3.9+
- ffmpeg (installed system-wide)

## Installation
pip install -r requirements.txt

## Usage
python main.py

## Notes
- Instagram downloads work for public posts only
- YouTube Shorts are supported
- No metadata or extra files are saved
